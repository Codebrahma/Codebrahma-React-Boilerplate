# Codebrahma-React-Boilerplate

A React Boilerplate with all the latest packages.

#### 1. React V16.8.4

#### 2. Webpack 4.29.6

#### 3. Style using SASS / CSS

#### 4. Code spliting with lazy and suspense

#### 5. Uglification & URL hashing for js builds

### For Installing

```javascript
npm install
```

### Linting

```javascript
npm run lint
```

### Building

```javascript
npm run build
```
